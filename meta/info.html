Beaglebone Black

cape Board

This project template is the basis of an expansion board for the Beaglebone Black

This base project includes a PCB edge defined as the same size as the Beaglebone Black PCB with the connectors placed correctly to align the two boards. All IO present on the Beaglebone Black board is connected to the project through the 0.1" expansion headers.
